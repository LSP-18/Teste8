# Ideias de Design - Shopee Backlog Carandiru

## Abordagem Escolhida: Dashboard Operacional Shopee

**Design Movement:** Operational Dashboard com identidade visual Shopee
**Core Principles:**
- Fidelidade à identidade Shopee (laranja #EE4D2D como cor primária)
- Clareza operacional: dados sempre visíveis e legíveis
- Formulário de entrada rápida e intuitiva
- Persistência local transparente (localStorage)

**Color Philosophy:**
- Primária: Shopee Orange #EE4D2D
- On Hold: Azul #0055AA
- Amarelo: #F5C518
- Erro: #EE4D2D
- Fleet: #26AA99
- Office: #64748B
- Background: #F6F6F6

**Layout Paradigm:**
- Sidebar esquerda com navegação
- Área principal com abas: Resumo Semanal | Adicionar Dados | Histórico

**Signature Elements:**
- Borda inferior laranja nos cards de header
- Badges coloridos por categoria
- Tabela com linha de total destacada

**Interaction Philosophy:**
- Formulário com validação em tempo real
- Toast de confirmação ao salvar
- Dados persistidos no localStorage automaticamente

**Typography System:**
- Inter (já usado nos HTMLs originais)
- Pesos: 400, 500, 600, 700, 800
